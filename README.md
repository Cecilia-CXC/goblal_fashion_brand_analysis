# goblal_fashion_brand_analysis
Code for data pre-processing for a dashboard on global fashion brands 

# Dataset used
https://www.kaggle.com/datasets/jocelyndumlao/global-fashion-brands 
Source: Kaggle 
Author: Jocelyn Dumlao

# About the Project 
This project involves looking at different fashion brands that are known globally and look at their rankings, growth rate and equities over time from 2001-2021.
This code is of the data pre-processing done to clean and transform the raw dataset to one that can be used to analse and display in a dashboard 

# How to run the project 
1. Download the dataset from Kaggle:

2. Open the notebook in Google Colab:
   - Upload `Fashion Brand Analysis.ipynb` to Colab  

3. Upload the downloaded dataset into Colab 

4. Run all cells from top to bottom.
